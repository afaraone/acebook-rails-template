## Instructions

Clone this repo:
```
$ git clone git@github.com:darciew/acebook-rails-template.git
$ cd acebook-rails-template
```

Install gems:
```
$ bundle install
```

Create database and run migration:
```
$ bin/rails db:create
$ bin/rails db:migrate
```

Run the test suite:
```
$ bundle exec rspec
```

Run server and visit https://localhost:3000
```
$ bin/rails server
```
